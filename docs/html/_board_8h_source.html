This report documents the development of our teamâ€™s text based C++ Fantasy Adventure Game, implemented using object-oriented programming principles, smart pointers, inheritance and polymorphism and probability mechanism. The main goal was to design a playable game where the player moves around a dynamic board, collecting items and fighting enemies while trying to earn as much gold as possible. Features such as character races with different abilities, a day/night cycle and item based stat modifications along with a probability based combat were all built into the system. Overall the aim is to design a clean and modular structure that makes the game easy to understand, extend and manage.
