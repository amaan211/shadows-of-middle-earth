The main goal was to design a playable game where the player moves around a dynamic board, collecting items and fighting enemies while trying to earn as much gold as possible. Features such as character races with different abilities, a day/night cycle and item based stat modifications along with a probability based combat were all built into the system. Detailed Description

Concrete class representing Elf characters.

Elfs have high attack but low defence and special ability: Successful defences always increase health by 1.
